## SECTION 1 — APP LOGO DESIGN PROMPT

### Logo Prompt (Primary — Horizontal)

```
Professional app logo for "PocketOS" — a personal mobile operating system app.

Design concept: A stylized terminal window merged with a circuit board and OS symbol. The word "PocketOS" rendered in a sharp, modern geometric sans-serif typeface (similar to Eurostile or Rajdhani Bold). Letters have a subtle cyan neon glow edge effect.

Left of the text: A minimalist icon showing a terminal command prompt symbol ">_" enclosed in a thin rounded-rectangle frame (like a terminal window titlebar). The frame has a subtle cyan (#00E5FF) inner glow. Three tiny dots (red, yellow, green) in the top-left corner of the frame for realism.

Color palette:
- Background: Deep space black (#0A0A0F)
- Primary text: Pure white (#E8E8F0)
- Accent glow: Electric cyan (#00E5FF)
- Secondary accent: Matrix green (#00FF88) for the ">_" symbol only
- Subtle gradient: Dark navy to pitch black in background

Style: Cyberpunk-minimal. Clean, premium, professional. NOT cartoonish. Similar aesthetic to Kali Linux branding but more refined and consumer-friendly. Flat design with subtle glow effects.

Typography feel: Technical, monospace-adjacent but not literally monospace. Sharp angles, high legibility, futuristic.

Format: Horizontal lockup, 2:1 aspect ratio, transparent background.
```

### Logo Prompt (Icon — Square App Icon)

```
Mobile app icon for "PocketOS" — a personal mobile operating system.

Shape: Perfect square with heavily rounded corners (iOS/Android style icon shape), 1024×1024px equivalent.

Design: A dark deep-space background (#0A0A0F to #070710 radial gradient, darker at edges). Centered: a stylized terminal prompt symbol ">_" rendered in bold electric cyan (#00E5FF) with a strong neon glow/bloom effect radiating outward. The glow has multiple layers: a tight bright core, a medium cyan bloom, and a faint outer halo reaching the icon edges.

Around the ">_" symbol: extremely subtle circuit trace lines in very dark cyan (#003340), barely visible, like a PCB motherboard pattern etched into the background. These are decorative only — not distracting.

Below the symbol, in very small JetBrains Mono style text: "POCKET OS" in dim cyan (#0097B2), tracking (letter spacing) wide.

Bottom-right corner: a tiny version indicator dot "v1" in barely visible grey.

Overall feel: Premium, dark, technical, mysterious. Like the Kali Linux dragon icon but minimalist and symbol-based rather than illustrated. High contrast. Instantly recognizable at 48px size.

No gradients on the symbol itself — pure flat cyan with glow only.
Style: Vector-clean, modern OS aesthetic, professional dark theme app icon.
```

### Concept C — Abstract Circuit OS Symbol

```
Minimal geometric logo mark for a mobile OS app called "PocketOS".

Colors: Symbol in gradient from #00E5FF (cyan, top) to #00FF88 (green, bottom). Text in #E8E8F0 white.

Ultra-clean. Vector art quality. Scalable from favicon to billboard.
Style: Think Linear.app or Vercel logo quality — pure geometric minimalism.
```

## SECTION 3 — APP WALLPAPERS

### Wallpaper 1 — "Boot Sequence" (Primary Default Wallpaper)

```
Ultra-premium dark mobile wallpaper for a personal OS app, portrait orientation, 1080×2340px.

Concept: Looking down a dark server room corridor rendered as an abstract digital space. 

Composition:
- Upper 60%: Deep space black background with extremely fine cyan (#00E5FF) horizontal scan lines at 2px intervals, opacity 3% — barely visible, like a CRT monitor.
- Dead center of the wallpaper: A glowing terminal window frame floating in space. The frame is a thin rounded rectangle border in cyan (#00E5FF) with 40% opacity and a very soft glow. Inside the frame, very small monospace text (barely readable) showing a boot sequence:
  "Initializing PocketOS..."
  "Loading kernel modules... [OK]"
  "System ready."
- Lower 30%: A deep dark gradient into pure black (#000000).
- Scattered across the entire image: 60-80 tiny dots of varying sizes (1-3px), some in cyan, some in white, some in green — like distant stars or floating particles. Distributed irregularly, denser in center-left.
- Vertical edge: On the right edge, a very thin (2px) vertical line of pure cyan that runs 40% of the wallpaper height from the bottom, with a bright point at the top that fades to nothing below.
- Bottom 15%: Extremely subtle circuit board trace pattern in #0D1520 (barely visible dark navy on black).

Color temperature: Cool. Entirely cold blues, cyans, greens. No warm tones.
Mood: Space station at 3am. Calm power. Silent intelligence.
Style: Kali Linux + ParrotOS wallpaper quality. Premium mobile dark wallpaper.
Format: Portrait 9:19.5, suitable for phone lockscreen/homescreen.
```

### Wallpaper 2 — "Neural Grid" (Alternative)

```
Premium dark mobile wallpaper, portrait orientation 1080×2340px, for "PocketOS" personal operating system.

Visual concept: An infinite perspective grid receding into a dark vanishing point, viewed from slightly above and ahead, like flying over a circuit board at night.

Grid properties:
- Grid lines: Very thin (1px), in deep cyan (#003344) — subtle, not bright
- Grid spacing: Appears to converge at a center-high vanishing point
- At grid intersections: Small glowing nodes. Most are dim white (opacity 20%), but every 8th intersection has a brighter cyan node (#00E5FF at 60%) that pulses faintly
- The grid occupies the bottom 70% of the wallpaper, appearing to extend below

Upper 30%:
- Pure dark sky (#070710)
- "PocketOS" logo mark (the >_ terminal symbol) floating centered at 25% height from top, rendered in pure cyan with a large soft glow bloom effect (200px radius soft glow), like a sun or moon in darkness
- Below the logo: tiny text "v1.0" in dim cyan, barely visible

Atmospheric effects:
- A very faint diagonal light beam crossing from top-left to bottom-right at 8% opacity, like a headlight in fog
- Bottom corners darker (vignette)
- Slight chromatic aberration on the logo mark only (tiny red/blue fringe, 1-2px)

Color palette: Strictly cool. #0A0A0F, #00E5FF, #003344, #FFFFFF at low opacity.
Style: Tron: Legacy meets Kali Linux. Epic but tasteful. High end gaming/OS aesthetic.
```

### Wallpaper 3 — "Matrix Rain" (Hacker Green Theme)

```
Dark mobile wallpaper, portrait 1080×2340px, green terminal aesthetic for "PocketOS" app (green theme variant).

Concept: Vertical columns of falling characters, but not the clichéd Matrix style — refined, minimal, premium version.

Character rain:
- Very sparse — only 8-10 thin columns across the width
- Characters: Mix of binary (0,1), single digits, and terminal symbols ( > _ { } [ ] )
- Color: Each character column starts bright green (#00FF88) at top and fades to nothing within 15-20% of screen height
- Character size: 10sp, monospace
- Columns fall at different speeds/offsets so they're staggered
- Most characters at 10-30% opacity — majority invisible, only a few bright
- One or two "leading" bright characters per column at full brightness

Background: Pure #050A05 (near-black with slight green tint, warmer than the cyan wallpaper)

Centerpiece: At 35% from top, centered: The PocketOS ">_" symbol in a larger size (150px equivalent), in #00FF88 bright green, with an intense bloom/glow effect. The glow is green (#00FF41) with outer corona fading to transparent.

Bottom 20%: Extremely dim circuit trace pattern, green on near-black.

Top 10%: Completely dark, no elements.

Overall mood: The green hacker terminal aesthetic. Like a terminal from a 1990s cyberpunk film but ultra HD and premium. Clean, not chaotic.
```

### Wallpaper 4 — "Kali Blue" (Cyber Blue Theme)

```
Premium dark wallpaper, portrait 1080×2340px, for PocketOS "Cyber Blue" theme.

Main concept: Abstract deep ocean floor meets digital system architecture.

Background layers (front to back):
1. Foreground layer: Three large overlapping translucent hexagons, each with thin (#4488FF blue at 30% opacity) borders, no fill. Arranged asymmetrically — one large centered, one small upper-right, one medium lower-left. Each has a subtle inner glow.
2. Mid layer: Fine hexagonal grid pattern in #0A1428 (dark navy, barely visible), covering entire background.
3. Back layer: Radial gradient from #0A0F1A (center) to #04060E (edges).

Accent elements:
- One bright line of cyan-blue (#4488FF) connecting two hexagon vertices, like an active network connection, with a tiny glowing node dot at each end
- 40-50 particle dots scattered randomly, most very dim (#ffffff at 5-15%), 3-4 of them bright blue (#4488FF at 80%)
- In the dead center: PocketOS icon (">_" symbol) in #4488FF with a wide soft glow

Text (very bottom, 5% from bottom edge):
"POCKETOS PERSONAL OS" in JetBrains Mono, 10sp, #4488FF at 40% opacity, letter-spacing 0.3em

No warm tones anywhere. Cold blue throughout.
Style: Professional cybersecurity / enterprise OS aesthetic. ParrotOS meets Vercel dark mode.
```

## SECTION 4 — SPLASH SCREEN ASSET PROMPT

### Splash Screen Center Graphic

```
App splash screen logo asset for "PocketOS" — designed to display centered on a pure #0A0A0F background.

Asset: The ">_" terminal prompt symbol, large format.
- Size: Approximately 120×80 points equivalent
- Symbol: ">_" — the ">" is slightly larger than the "_"
- Rendering: Perfect vector, sharp edges
- Color: Pure #00E5FF (electric cyan)
- Glow effect: Three-layer bloom:
  · Inner tight glow: 4px blur, #00E5FF at 100%
  · Middle glow: 12px blur, #00E5FF at 60%  
  · Outer halo: 30px blur, #00E5FF at 25%
- Below the symbol: "POCKETOS" in Rajdhani ExtraBold or similar, 18sp, white, letter-spacing 0.25em
- Below that: "v1.0" in JetBrains Mono, 10sp, #8888A0 (dim grey)

This will sit on top of the dark background. The entire asset including glow must render correctly on #0A0A0F.

Format: Transparent PNG, 600×400px.
Quality: App Store screenshot quality. No compression artifacts.
```

## SECTION 5 — USAGE GUIDE

### Which prompt to use where:

| Asset | Use Prompt | Target Size |
|-------|-----------|-------------|
| Play Store icon | Logo Prompt — Square App Icon | 512×512px |
| App launcher icon | Logo Prompt — Square App Icon | 192×192px |
| About screen logo | Logo Prompt — Horizontal | 800×200px |
| Default wallpaper | Wallpaper 1 — Boot Sequence | 1080×2340px |
| Green theme wallpaper | Wallpaper 3 — Matrix Rain | 1080×2340px |
| Blue theme wallpaper | Wallpaper 4 — Cyber Blue | 1080×2340px |
| Splash center asset | Splash Screen Center Graphic | 600×400px |
| Play Store feature graphic | Wallpaper 1 cropped 16:9 | 1024×500px |

### Generation Tips for Google ImageFX / Imagen 3:

1. Add `"--style photorealistic"` for more detailed results
2. Add `"--ar 9:19.5"` for portrait wallpaper ratio
3. For icon: add `"centered composition, subject fills 60% of frame, sharp focus, 8k quality"`
4. For wallpapers: add `"cinematic lighting, volumetric glow, ultra detailed, no text, no watermark"`
5. Generate 4 variations per prompt, pick best
